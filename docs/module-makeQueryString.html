# async inner makeQueryString(params) → {Promise.<string>|Error}

Utility method to generate a query string.
Prepend it to an API URL string.

Parameters:
Name Type Description
params object

Object of Type QueryStringParameters

View Source utils/makeQueryString.ts, line 6

An API query string

  • On resolve: Promise(<string>)
  • On reject: new Error(<string>) - if parameters are not of type 'object'
Promise.<string> | Error
Examples

Single Param:

const qs = await genQueryString({
  format: 'json'
}).catch(error => error)
//  qs = "format=json"

Multiple Params:

const qs = await genQueryString({
  format: 'json',
  modelYear: 2006,
  page: "2"
}).catch(error => error)
// qs = "?format=json&modelYear=2006&page=2"

Empty Params Object:

const qs = await genQueryString({}).catch(error => error)
// qs = ""
