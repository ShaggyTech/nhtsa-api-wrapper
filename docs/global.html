# isValidType

Validate types and existence of various input values.
Note: Uses Object.prototype.toString.call(value) under the hood

Returns true if:

  • type of value is equal to provided type

Returns false if:

  • type arg is missing
  • type or value arg is falsey or missing

Examples

Single Param:

const qs = await genQueryString({
  format: 'json'
}).catch(error => error)
// => Promise.resolve("?format=json")

Multiple Params:

const qs = await genQueryString({
  format: 'json',
  modelYear: 2006,
  page: "2"
}).catch(error => error)
// => Promise.resolve("?format=json&modelYear=2006&page=2")

Empty Params Object:

const qs = await genQueryString({}).catch(error => error)
// => Promise.resolve("")
