utils/genQueryString

Generates a query string for use with the NHTSA.gov API base URL

Example return: "?format=json&modelYear=2019"


Parameters:
Name Type Description
params object

An object containing key<ParamName> : value<String> api query parameters to use.

View Source utils/genQueryString.js, line 1

An API query string
On resolve: Promise(<string>)
On reject: new Error(error<string>)

Promise.<string> | Error
Examples

Single Param:

const qs = await genQueryString({
  format: 'json'
}).catch(error => error)
// => Promise("?format=json")

Multiple Params:

const qs = await genQueryString({
  format: 'json',
  modelYear: 2006,
  page: "2"
}).catch(error => error)
// => Promise("?format=json&modelYear=2006&page=2")

Error Handling - invalid params will reject with an Error: 

// No params provided; produces an error which is caught and returned as is
const qs = await genQueryString().catch(error => error)
// => error<Error>
