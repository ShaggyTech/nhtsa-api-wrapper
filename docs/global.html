# async utils/isValidType(options) → {Promise.<Boolean>}

Validate types and existence of various input values.
Note: Uses Object.prototype.toString.call(value) under the hood

Returns true if:

  • type of value is equal to provided type

Returns false if:

  • 'options.value is not of type options.type
  • 'options.type' is missing from options object
  • 'options.type' is not of type 'string'
  • 'options.value' is falsey or not provided
Parameters:
Name Type Description
options utils/isValidType.IsValidTypeOptions

Single argument object containing 'type' and 'value' keys
type string

What type are you expecting the value to be?
value any

What value are we testing against?

View Source utils/isValidType.ts, line 2

Promise.<Boolean>
Examples

Verify type string

const isValid = await isValidType({
  type: 'string',
  value: 'this is a string'
}).catch(error => error)
//  isValid = "true"

Verify type object

const isValid = await isValidType({
  type: 'object',
  value: { someKey: 'some value' }
}).catch(error => error)
//  isValid = "true"

Returns false if type is not valid

const isValid = await isValidType({
  type: 'array',
  value: 'this is not an array'
}).catch(error => error)
//  isValid = "false"

# async utils/makeQueryString(params) → {Promise.<string>|Error}

Utility method to generate a query string.
Prepend it to an API URL string.

Parameters:
Name Type Description
params utils/makeQueryString.QueryStringParameters

An object containing key:value API query parameters to use.
Parameter values may be either type 'string' or type 'number'

View Source utils/makeQueryString.ts, line 2

An API query string

  • On resolve: Promise(<string>)
  • On reject: new Error(<string>) - if parameters are not of type 'object'
Promise.<string> | Error
Examples

Single Param:

const qs = await genQueryString({
  format: 'json'
}).catch(error => error)
//  qs = "format=json"

Multiple Params:

const qs = await genQueryString({
  format: 'json',
  modelYear: 2006,
  page: "2"
}).catch(error => error)
// qs = "?format=json&modelYear=2006&page=2"

Empty Params Object:

const qs = await genQueryString({}).catch(error => error)
// qs = ""
