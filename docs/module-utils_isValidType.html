# static isValidType(options) → {boolean}

Provided an object with type and value keys and it will:

Return true if:

  • type of value is equal to provided type.

Return false if:

  • value is not equal to provided type
  • type is missing from options object
  • type is not of type string
  • value is falsey or otherwise not provided.
Parameters:
Name Type Description
options object

IsValidTypeOptions.
type string

What type are you expecting the value to be?
value any

What value are we testing against?

True or false.

boolean
Examples

Verify type string

const isValid = isValidType({
  type: 'string',
  value: 'this is a string'
}).catch(error => error)

console.log(isValid) // "true"

Verify type object

const isValid = isValidType({
  type: 'object',
  value: { someKey: 'some value' }
}).catch(error => error)

console.log(isValid) // "true"

Returns false if type is not valid

const isValid = await isValidType({
  type: 'array',
  value: 'this is not an array'
}).catch(error => error)

console.log(isValid) // "false"
